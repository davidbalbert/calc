calc
====

A simple calculator made with lex and yacc. Trying to make it feature compatable with [jisoncalc](http://github.com/davidbalbert/jisoncalc).

##Using

```
$ make
$ echo '5 + 5' | ./calc
10
$ ./calc
>> foo = 2 * 3
=> 6
>> bar = foo ^ 2
=> 36
>> baz = foo + bar
=> 42
```
